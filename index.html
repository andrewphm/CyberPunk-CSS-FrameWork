Paragraph styles

This is the strong tag in a paragraph. This is the em tag in a paragraph, elements such as <strong> and <em> are inline. Here is an example of the abbreviation tag. You'd use abbr for things like HTML and CSS.

				
	<p>This is an <strong>average</strong> paragraph.
	It includes some <em>inline</em> elements such as <strong> and <em>.</p>

Buttons

This is your average button. You might want to click it.


		
	<button>Submit</button>

Custimize buttons by adding CSS variables inline or with a classes.

This button has consequences. You might want to think twice before clicking.

There is also an effect when you hover over buttons.

					
	<button class="danger">Delete</button>
